# hireme.scrapper

Before the start:
1. Create python virtualenv and install requirements
2. Start redis server
3. Edit or create `.env` file with the next fields:

* DB_USER
* DB_PASSWORD
* DB_HOST
* DB_PORT
* DB_NAME
* REDIS_PROT
* REDIS_HOST
* REDIS_PORT
* REDIS_DB
* SCRAPPER_HOST
* SCRAPPER_PORT
* SCRAPPER_URL
